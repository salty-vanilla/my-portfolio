## Project Overview

自動外観検査設備の実導入プロジェクト。検査アルゴリズムの実装から光学系設計、設備選定、工場への実導入まで一貫して担当しました。

## Your Role

- **Algorithm Development**: 外観検査アルゴリズムの設計と実装
- **Optical System Design**: 照明・カメラ配置の最適化
- **Equipment Selection**: カメラ、照明、PC等のハードウェア選定
- **Factory Deployment**: 製造現場への設備導入とオペレーション設計

## Technical Implementation

### Inspection Algorithm
- 画像処理による欠陥検出アルゴリズム
- 機械学習による分類モデルの構築
- リアルタイム処理の最適化

### Optical Design
- 照明条件の最適化
- カメラ配置・レンズ選定
- 撮像条件の標準化

### System Integration
- PLC連携による設備制御
- 検査結果のデータベース記録
- アラート・通知システムの構築

## Impact & Results

- **検査時間**: 70%削減（従来の目視検査比）
- **検査精度**: 99.5%の欠陥検出率を達成
- **生産性向上**: 検査工程のボトルネック解消
- **品質安定化**: 人的ミスの削減、検査基準の標準化

## Technologies Used

- **Image Processing**: OpenCV, NumPy, scikit-image
- **Machine Learning**: Python, TensorFlow, scikit-learn
- **Hardware**: Industrial cameras, LED lighting systems
- **Integration**: PLC communication, database systems
